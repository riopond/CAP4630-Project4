The GitHub repository is intended to hold our data file that we link to our Google Colab Python notebook.

Google Drive (drive.google.com) is needed to use Google Colab. Open a tab in your browser and go to https://colab.research.google.com/drive/1jHhwRyVjfudhMEMNOLezY-N2uMGXJDUI?usp=sharing

To run this program, either go to Runtime and press the "Run All" button, or run each cell in order, starting with the first cell containing the imports.

If you would like to change the training-testing split, go to Cell 11 (labeled with a comment about setting the aforementioned sets) and adjust the test_size value.

Observe the error and accuracy results in the respective sections of the code for each learning model.
